File Upload

To ensure successful upload, please note the following guidelines:
  1. File Format: Only files with the extension .csv are accepted.
  2. Required Columns: Your CSV file must contain the following columns: 'Category Name', 'Counter ID', 'Operator Name', 'Wait Time', and 'Service Time'.
  3. Preparing Excel Files: If your data is in an Excel file, please convert it to CSV format. Before conversion, ensure that the first row of your Excel sheet contains the column names. If the first row contains any other data, such as titles or timestamps, please remove it to avoid potential errors during conversion.

Following these guidelines will help streamline the upload process and ensure accurate data integration. If you have any questions or encounter any issues, feel free to reach out for assistance.
